# Release Notes: v0.3.0

## Highlights

- Added CTID Mappings Explorer JSON support for ATT&CK-aware vulnerability context.
- Added local ATT&CK technique metadata loading with tactic resolution and lifecycle flags.
- Upgraded `analyze`, `compare`, and `explain` with visible ATT&CK context and run summaries.
- Added `attack validate`, `attack coverage`, and `attack navigator-layer`.
- Added pinned ATT&CK fixtures, sample inputs, and example artifacts for the V0.3 workflow.

## Upgrade Notes

- The primary priority score is still based on CVSS, EPSS, and KEV.
- ATT&CK remains optional and local-file based.
- `--offline-attack-file` still works for the legacy CSV path.
- `--attack-source ctid-json` is the preferred V0.3 ATT&CK workflow.
