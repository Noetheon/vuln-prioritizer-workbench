# Release Notes: v0.2.0

## Highlights

- Added `compare` for deterministic `CVSS-only vs enriched` analysis.
- Added post-enrichment filtering and sort options to `analyze`.
- Added configurable policy thresholds for enriched prioritization.
- Upgraded `explain` with baseline comparison output and optional ATT&CK context.
- Added a slim GitHub Actions workflow that mirrors the local `make check` gate.

## Upgrade Notes

- The default prioritization behavior is unchanged unless policy override flags are used.
- The ATT&CK workflow remains optional and offline-file-based.
